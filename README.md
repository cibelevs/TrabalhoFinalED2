Sistema de Gerenciamento de Voos

Este é um projeto de um sistema de gerenciamento de voos desenvolvido utilizando Python e Flask. O objetivo é criar uma aplicação web simples para gerenciar informações sobre voos, como itinerários, horários e passageiros.


Funcionalidades

A aplicação oferece as seguintes funcionalidades (dependendo do que seu código faz, você pode personalizar esta parte):

Cadastro de voos: Permite o cadastro de novos voos no sistema, incluindo informações como origem, destino e horários.

Consulta de voos: Exibe uma lista de todos os voos cadastrados, com detalhes.

Edição de voos: Permite editar informações de voos existentes.

Remoção de voos: Você pode excluir voos que não são mais necessários.


Pré-requisitos

Antes de rodar a aplicação, é necessário garantir que o ambiente de desenvolvimento esteja configurado corretamente.

Ative o ambiente virtual: 
- no linux: source venv/bin/activate
- no windows: venv\Scripts\activate

Com o ambiente virtual ativado, instale as dependências do projeto (no caso, o Flask): pip install flask

Agora você pode rodar o servidor da aplicação Flask com o seguinte comando: python3 sistemaVoos.py
